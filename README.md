### Multi-probe Manipulation Rig

Neural probe manipulation requires precise manipulation to avoid errors and optimize good placement for signal development. Janelia researchers and the Janelia Experimental Technology group have developed a multi-probe manipulation rig for such experiments with rodents. The base surface is magnetic stainless steel and the manipulator towers can be freely moved then locked in place with magnetic clamps for rough positioning. Sensapex 3 axis micromanipulators then provide precise positioning. Adapter designs for the manipulator include holders for Neuropixels probes.
